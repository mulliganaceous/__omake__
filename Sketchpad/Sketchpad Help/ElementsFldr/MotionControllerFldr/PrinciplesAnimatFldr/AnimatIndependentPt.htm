Animation of an Independent Point

An independent point is animated by moving it about in the plane. The animation speed determines how far it’s likely to move in each random step. You cannot control the direction for an independent point. To specify the starting animation speed explicitly or to make the motion occur one time only, create an Animation action button that animates the point.

Merge independent points to paths if you want them to move in a specific direction.
