Using Circles and Arcs

·         Use a circle or an arc as a path along which to animate a point by creating the point with the Point On Object command.

·         Use a circle or an arc as a path along which to animate an existing point by using the Merge command.

·         Construct the Interior of a circle or the Arc Segment or Arc Sector interior of an arc.

·         Construct the Intersection of a circle or an arc with a straight object, a circle, or an arc.

·         Use Define Unit Circle to define a coordinate system that uses the selected circle as its unit circle.
