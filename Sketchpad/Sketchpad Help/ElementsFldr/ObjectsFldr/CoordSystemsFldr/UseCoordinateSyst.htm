Using a Coordinate System

·         To change the scale of a coordinate system, drag the unit point. Square coordinate systems have one unit point, and rectangular coordinate systems have two.

·         To change the scale of a coordinate system, press and drag any axis tick-mark label with the Arrow tool.

The scale of a coordinate system may be defined by an object in the sketch; such a coordinate system doesn’t have a unit point and cannot be scaled by dragging axis tick-mark labels.

·         If you have more than one coordinate system, make one of them the active one by choosing Mark Coordinate System.

·         To plot a point on the coordinate system, choose Plot Points.

·         To plot two measured values on the coordinate system, select them and choose Plot As (x, y).

·         To plot a new function on the coordinate system, choose Plot New Function.

·         To plot an existing function on the coordinate system, select the function and choose Plot Function.
