Elements

If you’ve worked through the Learning Guide, you already have a good sense of many of the elements of Sketchpad. Broadly speaking, you use Sketchpad to create documents containing mathematical diagrams and geometric figures. Each diagram or figure—each sketch—is constructed from individual objects that are defined in terms of their mathematical relationships to each other. You use a combination of tools and menu commands to interact with documents and the objects in them. While the next two top-level Help books describe each of these tools and commands in detail, this section introduces the various types of objects you’ll create and surveys the tools and commands you’ll use to manipulate them. Refer to this section when you encounter unfamiliar types of objects in Sketchpad or when you’re looking for properties of objects you’re already acquainted with.
