Scroll Button

This command is available only when a single point is selected.

It creates a button that scrolls the window based on the position of the selected point.

When you create a Scroll button, the Scroll panel of the Properties dialog box appears to allow you to determine whether the button scrolls the window to put the selected point at the top left corner of the window, or to put the selected point in the center of the window.

Use Scroll buttons when you want to be able to “jump” to a point anywhere in the scrollable plane.
