Plot Properties for a Locus

Use this panel to set the number of samples in a locus. If the locus is a point locus, you can also use this panel to determine whether the locus is displayed in continuous or discrete form. The illustration below shows an elliptical point locus displayed in continuous form on the left and in discrete form on the right.

     

You can adjust the number of samples in a locus or function plot without going to Properties by selecting the sampled object and pressing the + or – key while your sketch window is active.
