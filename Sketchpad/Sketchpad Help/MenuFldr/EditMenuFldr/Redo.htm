Redo

This command redoes an action you have undone. If you’ve undone several steps, you can redo each of those steps.

The keyboard shortcut for Redo is Ctrl+R (Windows) or z+R (Mac).

Redo is available only immediately after using Undo. If you take any other action after undoing operations, you can no longer redo the original operations.

Use Redo in combination with Undo to move backward and forward through your recent Sketchpad actions.

When you hold down the Shift key, Redo becomes Redo All and redoes all previously undone actions.
