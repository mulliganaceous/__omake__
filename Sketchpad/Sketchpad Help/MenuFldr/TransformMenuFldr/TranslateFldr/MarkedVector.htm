Marked Translation Vector

Choose this option to define the translation by specifying a vector’s initial and terminal points.

 

This option is enabled only if you’ve already marked a vector in your sketch. If you haven’t already used the Mark Vector command to mark a vector, you can mark one now by clicking two points in your sketch—first the initial point, then the terminal point.
