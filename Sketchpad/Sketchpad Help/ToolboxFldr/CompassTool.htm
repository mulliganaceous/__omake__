Compass Tool

Use the Compass tool to construct circles determined by two points—the center point and another point through which the circle passes. This second point is sometimes called the radius point, because it determines the radius of the circle.

Related Commands

There are two Construct menu commands that construct circles without using the Compass tool.

·         Circle By Center+Point constructs a circle determined by two selected points. The first point is the center point, and the second is the radius point.

·         Circle By Center+Radius constructs a circle determined by a selected point and a selected distance. The selected point is the center point, and the selected distance determines the radius.

The selected distance can be either a segment or a distance measurement.
