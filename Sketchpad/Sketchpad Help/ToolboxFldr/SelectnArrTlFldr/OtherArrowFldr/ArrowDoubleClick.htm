Arrow Tool Double-Click Shortcuts

Double-clicking the Arrow tool on several different kinds of objects quickly accomplishes the most common action for that kind of object. Double-clicking provides shortcuts for these actions that would otherwise require you to select the object and then choose a menu command.

Double-Clicked Object

Resulting Action

Point

Mark Center for rotations and dilations

Straight Object

Mark Mirror for reflections

Calculation

Edit Calculation

Function

Edit Function

Parameter

Allows you to change the parameter’s value

Label

Shows Label Properties to edit the label

Caption

Begins editing the caption

Table

Adds a row to the table
