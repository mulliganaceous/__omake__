Translate Arrow Tool

Drag selected objects with this tool to translate them—that is, to slide them by any distance in any direction without turning or changing size or shape. This is the default Arrow tool.

You can use the Translate Arrow tool to select and deselect objects, press action buttons, and construct points of intersection just like the other Selection Arrow tools.
